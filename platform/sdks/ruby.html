Momento Ruby SDK

The Momento Ruby SDK is available via the rubygems package momento.

The source code can be found on GitHub: momentohq/client-sdk-ruby.

Requirements​

  • A Momento API Key is required; you can get one from the Momento Web Console.
  • Ruby 2.7 or later.
