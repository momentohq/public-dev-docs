## Execution constraints
*Momento Functions* are designed for extreme throughput and low-latency execution. Each function runs in an optimized WebAssembly (WASM) environment, ensuring sub-50ms execution times at unlimited scale.

Functions are stateful within the execution environment, meaning they can maintain local in-memory data across invocations while still operating at high throughput.

## Integration with external services
*Momento Functions* support direct integrations with AWS services:

- **Amazon S3**: Functions can write files (`s3:PutObject`) and retrieve metadata (`s3:HeadObject`, `s3:ListBucket`).
- **Amazon Bedrock**: Functions can invoke AI models using `bedrock:InvokeModel`.

To configure AWS permissions, see [AWS Role Delegation](/functions/authentication/aws-role-delegation).
