# Momento Storage - Early Access Preview

**Momento Storage** reimagines the persistent key-value datastore by adding an
integrated cache and gateway layer. This consolidated design replaces multiple pieces
of a traditional cloud architecture with a single managed, enterprise-grade service
engineered for extremely high-volume workloads.

Contact us if you are interested in our early-access preview!
