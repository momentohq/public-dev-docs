# Momento とクォータと制限

:::info
サービスクォータはアカウントのサポートプランに直接影響されます。<ins>ソフトリミット</ins>とハードリミット</ins>の両方は、プロティアプランとエンタープライズプランで増やすことができます。増額をご希望の場合は、[営業担当までご連絡ください](https://www.gomomento.com/contact-us).
:::
