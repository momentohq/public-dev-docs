# Momento認証サービスのクォータ

:::info
サービスクォータは、アカウントのサポートプランに直接影響されます。<ins>ソフトリミットとハードリミット</ins>の両方は、プロティアプランとエンタープライズプランで増やすことができます。上限緩和をご希望の場合は、[営業担当までご連絡ください](https://www.gomomento.com/contact-us).
:::
