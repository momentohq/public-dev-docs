# RustでMomentoストレージを始める

RustとMomento Storageをすぐに使い始める必要がある場合、このページには必要な基本的なAPIコールが含まれています。[Rust SDK のサンプル](https://github.com/momentohq/client-sdk-rust/tree/main/examples) には、ビルド設定ファイルを含む完全なサンプルがあります。

## Momento SDKをインストールする

Momento SDKはcrates.io: [momento](https://crates.io/crates/momento)で入手できます。

:::tip
[crates.io](https://crates.io/crates/momento)にアクセスし、利用可能な最新バージョンのSDKを見つけてください。
:::

既存のRustプロジェクトにクライアントライブラリをインストールする:

```cli
cargo add momento
```

## APIキーの設定

Momentoで認証するには、Momento APIキーが必要です。 Momento Web Console](https://console.gomomento.com/api-keys)から取得できます。
APIキーを取得したら、Momentoクライアントが利用できるように環境変数に保存してください：

```
export MOMENTO_API_KEY=<your Momento API key here>
```
