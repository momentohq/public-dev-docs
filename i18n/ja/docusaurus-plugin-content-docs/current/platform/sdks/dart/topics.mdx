# DartでMomento Topicsを始める

DartとMomentoトピックをすぐに使い始める必要がある場合、このページには必要な基本的なAPIコールが含まれています。[Dart SDK のサンプル](https://github.com/momentohq/client-sdk-dart/tree/main/example) を参照してください。

## Install the Momento SDK

Momento Dart SDKは[pub.dev as `momento`](https://pub.dev/packages/momento)で入手可能です。

Dartプログラムにインストールするには、以下を使用します：

```bash
dart pub add momento
```

Flutterプログラムにインストールするには、次のようにします：

```bash
flutter pub add momento
```

## Momento APIキーを取得する

Momento APIキーが必要です。Momento Web Console](https://console.gomomento.com/)から取得できます。
トークンを取得したら、Momentoクライアントが利用できるように環境変数に保存します：

```
export MOMENTO_API_KEY=<your api key here>
```

**Note**: セキュリティ強化のためには、APIキーを環境変数ではなく、AWS Secret ManagerやGCP Secret Managerのようなものに入れるのがベストプラクティスだが、ここではデモのためにAPIキーを使っている。
