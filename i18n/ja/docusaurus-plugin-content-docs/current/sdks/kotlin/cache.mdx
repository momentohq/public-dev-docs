# KotlinでMomentoキャッシュを始める

KotlinとMomento Cacheをすぐに使い始める必要がある場合、このページには必要な基本的なAPIコールが含まれています。ビルド設定ファイルを含む完全で実用的な例については、[Kotlin SDK examples](https://github.com/momentohq/client-sdk-kotlin/tree/main/examples) を確認してください。

## Momento SDKをインストールする

Momento SDKはMaven Centralで入手できます: [`software.momento.kotlin/sdk`](
https://central.sonatype.com/artifact/software.momento.kotlin/sdk)にあります。

:::tip
[Maven Central](https://central.sonatype.com/artifact/software.momento.kotlin/sdk)にアクセスして、SDKの最新バージョンを見つけてください。
:::

Install the client library in an existing Kotlin project:

### Gradle

```kotlin
implementation("software.momento.kotlin:sdk:x.x.x")
```

### Maven

```xml
<dependency>
    <groupId>software.momento.kotlin</groupId>
    <artifactId>sdk</artifactId>
    <version>x.x.x</version>
</dependency>
```

## APIキーの設定

Momentoで認証するには、Momento APIキーが必要です。 Momento Web Console](https://console.gomomento.com/caches)から取得できます。
APIキーを取得したら、Momento クライアントが利用できるように環境変数に保存します：

```
export MOMENTO_API_KEY=<your Momento API key here>
```
