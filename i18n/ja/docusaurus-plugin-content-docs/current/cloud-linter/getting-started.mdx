# はじめに
1. [Momento CLI](https://github.com/momentohq/momento-cli#installation)をインストールします。注意: クラウドリンターコマンドを実行するためにMomento認証トークンは必要ありません。
2. ターミナルで[AWS credentials](https://docs.aws.amazon.com/cli/latest/userguide/cli-configure-files.html)を設定します。
3. コマンド `momento preview cloud-linter` を実行します。
4. [オプション] 現在の作業ディレクトリにある `linter_results.json.gz` の生の出力を確認する。
5. [Momento Console](https://console.gomomento.com/cloud-linter)で `linter_results.json.gz` をアップロードします。
6. しばらくすると自動解析結果が表示されます。
